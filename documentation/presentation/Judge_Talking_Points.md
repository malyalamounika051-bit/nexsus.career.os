# Key Talking Points for Judges & Evaluators

## Purpose
A reference sheet for the team during post-demo Q&A to ensure key selling points are clearly communicated.

## Talking Points

### 1. The Single Source of Truth Concept
- **Message**: Nexus eliminates data duplication. Unlike other resume builders that save disconnected document text, Nexus uses a single, centralized MongoDB user profile to drive all features (mock interviews, resume tailoring, skill gaps).

### 2. High-Fidelity UI Customization
- **Message**: Resumes are rendered using clean, premium CSS templates with dynamic scaling controls, ensuring exports match standardized formats perfectly without clipping.

### 3. Practical AI Integration (No Hallucinations)
- **Message**: We implemented a programmatic verification step. While the AI optimizes resume phrasing for target roles, it is restricted to the user's verified profile data, preventing fabricated skills or credentials.
