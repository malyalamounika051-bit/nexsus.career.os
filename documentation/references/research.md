# Research References: ATS Parsing & Resume Matching

## Purpose
Academic and industry insights on modern hiring systems and automated resume screening tools.

## Key Insights
- **ATS Systems Overview**: Standard ATS parsers (like Workday, Taleo) look for exact matching keyword strings in resume sections. Simple layouts with clean headers perform best.
- **The Role of Action Verbs**: Strong action verbs at the start of bullet points improve readability scores. Quantifying outcomes (e.g., "reduced latency by 40%") improves matching rates by 2.2x.
- **Tailoring Effectiveness**: Tailored resumes receive up to a 60% higher response rate compared to generic profiles, underscoring the value of single-profile dynamic tailoring.
