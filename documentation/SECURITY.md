# Security Policy

## Supported Versions
We actively patch and maintain the current major release version of Nexus Career OS:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability
Do **not** open a public issue for security bugs. Instead, email reports to `security@nexusos.dev`. 

We will acknowledge receipt within 24 hours and provide an update on remediation plans within 72 hours. Once resolved, we will publish a security advisory and credit the reporter.
