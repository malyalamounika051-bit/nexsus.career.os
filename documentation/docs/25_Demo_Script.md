# Demo Script: Hackathon Live Flow

## Purpose
A chronological guide for presenting the live demo, outlining talking points, screen views, and user transitions.

## Demo Flow

| Duration | Screen View | Presenter Script / Action | Visual Highlights |
| --- | --- | --- | --- |
| **00:00 - 00:30** | Command Center | "Welcome! Today we are introducing Nexus Career OS. The single profile that automates your tech job search..." | Clean, modern dashboard metrics. |
| **00:30 - 01:15** | Profile Editor | "We start with the Profile—the single source of truth. Users add education, skills, and projects once..." | Show verified profile skills (e.g. JavaScript, Python). |
| **01:15 - 02:00** | Resume Builder | "When applying to jobs, copy-pasting is gone. By entering a job description, our AI tailors the resume..." | Show AI compiling summary, experiences, and projects. |
| **02:00 - 02:30** | Skill Gap Panel | "Real-time verification lists matching vs missing skills, providing a clear roadmap of what to learn next..." | Point out missing skills and target ratings. |
| **02:30 - 03:00** | Template Export | "With a click, we switch visual styles and export a ready-to-submit, recruiter-friendly PDF." | Swap templates and trigger the PDF export download. |
