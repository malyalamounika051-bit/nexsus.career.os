# Future Scope: Post-Hackathon Vision

## Purpose
Specifies product ideas, integrations, and system additions for future roadmap iterations.

## Future Plans

### 1. Direct Recruiter Dashboard
- **Goal**: Enable verified companies to search and sort anonymous candidate profiles based on verified project metrics.
- **Benefit**: Connects candidates directly with recruiters, bypassing traditional application pipelines.

### 2. Multi-modal AI Mock Interviews
- **Goal**: Expand the text-based SARA AI mentor into real-time audio/video mock interview simulations.
- **Benefit**: Provides live feedback on communication style, eye contact, and pacing.

### 3. Automatic Portfolio Syncing
- **Goal**: Auto-verify code contributions by directly monitoring GitHub repositories and mapping them to profile project metrics.
- **Benefit**: Eliminates the need for manual project updates, creating a dynamic showcase of candidate coding skills.
